# FitPlanHub - Fitness Plans Platform

FitPlanHub is a comprehensive fitness platform connecting certified trainers with fitness enthusiasts. Trainers can create and sell fitness plans, while users can browse, subscribe to plans, and follow their favorite trainers.

## Features

### For Users:
- Browse and preview fitness plans
- Subscribe to plans (simulated payment)
- Follow/unfollow trainers
- Personalized feed with plans from followed trainers
- User profile with subscription management

### For Trainers:
- Create, edit, and delete fitness plans
- Dashboard with subscriber statistics
- View list of subscribers
- Track revenue and plan performance

## Tech Stack

**Frontend:**
- React 18.2
- React Router DOM 6.20
- Vite 5.0
- CSS3 (Custom CSS)
